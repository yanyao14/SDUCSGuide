## 志愿服务
1.  校内校外都有志愿活动，具体可以看学校公众号推文。
基本每周都有的：实验农场管理（公众号问卷报名）、生命沙洲（qq群报名）、敬老院服务（qq群报名）、助学活动（qq群报名）等。（欢迎补充~）
在学校里不定时有大型活动（学术研讨会之类的），参加这种志愿时长很多。也可以加入志愿者储备队qq群（600906557），里面会有很多志愿者招募。
